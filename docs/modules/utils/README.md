# Módulo `utils`

## Objetivo
Funções utilitárias de apoio à aplicação.

## Arquivos
- `AppError.js`: erro de domínio com status HTTP.
- `asyncHandler.js`: wrapper para controllers assíncronos.
- `finance.js`: utilitários numéricos (`toNumber`, `round2`).

## Observação
Tudo aqui deve ser genérico e sem dependência de camada HTTP específica.
