# Política de Segurança

Obrigado por ajudar a manter o SobraRota seguro.

## Como reportar uma vulnerabilidade
Envie um email para:

- freitasalbert4@gmail.com

Inclua no reporte:
- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto esperado
- Prints ou logs, se houver
- Sugestão de correção, se possível

## Escopo
Este projeto inclui:
- Backend
- API REST
- Autenticação
- Banco de dados

## Boas práticas aplicadas
- Senhas com bcrypt
- Autenticação com JWT
- Validação de entrada no backend (Yup)
- CORS restrito
- Helmet
- Rate limit em autenticação
- Tratamento seguro de erros
- Variáveis sensíveis fora do repositório
